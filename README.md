# 3D Graphics Study

### 이 프로젝트의 목적
* 3D 모델이 어떤 과정을 거쳐 모니터에 출력되는지 원리를 이해하기 위함
* 이론을 _직접_ 이해하고, 중간에 계산 과정이 있을 경우 _직접_ 계산을 수행해서 공식을 도출한다.
* 이 프로젝트에서 프로그래밍은 이론 및 계산 결과에 대한 검증용으로 사용한다.
  * 만약 계산 결과가 잘못되었다면 프로그램에서 정상적으로 출력되지 않을 것이다.
